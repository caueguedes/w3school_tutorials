Position Property

Different positions:

  • static
  • relative
  • fixed
  • absolute
  • sticky

Static

This div element has position: static
Static elements are not affected by top/left/right/bottom properties It's always positioned with the normal page flow

Relative

This div element has position: relative
Setting top/right/left/bottom properties will cause the element adjust away from its position Other elements will not be adjusted to fit into any gap left

Fixed

This div element has position:fixed
The top/right/left/bottom properties are positioned relative to the viewport The fixed element does not affect the page flow

Absolute

This div element has position:absolute
The element is positioned relative to the nearest positioned ancestor and if it has no positioned ancestors, it uses the document body

Sticky

This div element has position:sticky
An element Sticky is positioned based on the users scroll position It will sticky when reach the determined position
